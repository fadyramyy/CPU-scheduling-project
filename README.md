# Project Overview
This project implements and compares two CPU scheduling algorithms:

- Round Robin (RR)
- Shortest Job First – Non-Preemptive (SJF)

The simulator computes:
- Waiting Time (WT)
- Turnaround Time (TAT)
- Response Time (RT)
- Average metrics
- Gantt Charts
- Comparison summaries
